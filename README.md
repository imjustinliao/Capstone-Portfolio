# Capstone Portfolio

A minimal, black-and-white portfolio site for the CS Capstone at the University of Washington.

## Pages

| Page                   | File                          |
| ---------------------- | ----------------------------- |
| Home (Landing)         | `index.html`                  |
| The User               | `user.html`                   |
| Capstone Experience    | `capstone-experience.html`    |
| Annotated Bibliography | `annotated-bibliography.html` |
| Overall Capstone       | `overall-capstone.html`       |

## Run Locally

No build tools required. Open `index.html` in any browser:

```bash
open index.html        # macOS
# or just double-click index.html in Finder / Explorer
```

## Deploy to GitHub Pages

1. Push the repository to GitHub.
2. Go to **Settings > Pages**.
3. Set the source to the **main** branch, root (`/`) folder.
4. The site will be live at `https://<username>.github.io/Capstone-Portfolio/`.

## What to Edit Later

- Replace placeholder text in each HTML file's `<main>` section with real content.
- Update the footer year or add links in the `<footer>` of each page.
- Add images to `assets/` and reference them as needed.
- All styling lives in `assets/css/styles.css`; all JS in `assets/js/main.js`.
